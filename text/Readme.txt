Системные требования
Перед установкой убедитесь, что ваша система соответствует следующим требованиям:
- Операционная система: Windows 10/11 (64-bit)
- Процессор: 2 ГГц или быстрее
- Оперативная память: 4 ГБ (рекомендуется 8 ГБ)
- Свободное место на диске: 500 МБ

Инструкция по установке

Для Windows:
1. Дважды кликните по файлу "BunkerInstaller"
2. Если появится окно контроля учетных записей, нажмите "Да" для разрешения внесения изменений
3. Следуйте инструкциям мастера установки:
   - Выберите язык интерфейса
   - Укажите путь установки (или оставьте значение по умолчанию)
   - При необходимости создайте ярлык на рабочем столе
   - Нажмите "Установить"
4. После завершения установки нажмите "Готово"

Первый запуск
После установки вы можете запустить приложение:
- Через ярлык на рабочем столе (если вы его создали)
- Из меню Пуск
- Через поиск в системе по имени приложения

Удаление приложения
Для удаления используйте стандартные средства вашей ОС.
